# The Seams Between Worlds
:octicons-info-24:{ .lg .middle } **A [Drankorian](<../../history/historical-realms/drankorian-empire.md>) cosmological treatise, written during the [Drankorian Era](<../../history/history-of-the-drankorian-empire.md>), by Ishara Venn**  

*The Seams Between Worlds* is Ishara Venn's systematic study of planar connections, especially [elemental vortexes](<../../cosmology/planar-concepts/elemental-vortex.md>). Ishara defines an elemental vortex as a temporary locus of connection created when a nearly pure representation of one of the Five Fundamental Energies draws an Energy Realm near to the [Material Plane](<../../cosmology/material-plane.md>).
