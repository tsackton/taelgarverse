# The Leandre
>[!info]+ Information
> 
>> 

A small river that runs from the  [Cleenseau Wood](<../../sembara/barony-of-aveil/cleenseau-region/cleenseau-wood.md>) to the [Auberonne](<./auberonne.md>). It is not navigable, and tends to be rocky and shallow.
